El explorador de postulantes para un imaginario puesto laboral administrativo, muestra la información del currículum de una serie de candidatos al mismo, permitiendo recorrer la lista de forma secuencial haciendo click en el menú "Siguiente".

La información de interés particular puede enfococarse mediante el uso del menú superior, y se muestra en controles tipo spoiler que permiten mostrarla u ocultarla a discreción del usuario, cuyos datos pueden capturarse en una sección de contacto.

Los datos personales y fotografía son obtenidos aleatoriamente desde la web de RamdomUser, y el resto del contenido fué generado desde la web de Loren Ipsum.
